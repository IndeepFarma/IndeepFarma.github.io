Projects

Galactic Travels

Galactic Travels is a flash game I made back in grade 10. It is inspired by old space themed games. The goal is to save the planet from asteroids that are speeding their way towards it. You catch all them with the spaceship which follows your mouse. Check out the game at https://www.kongregate.com/games/IndeepFarma/galactic-travels


Personal Website

My personal website (this website) I made from scratch using HTML and CSS, it has been a great learning experience creating a unique and fun user interface. I have utilized many features such as parallax imaging, slideshows, smooth scroll, and other features to keep the website clean and friendly. Programming the website helps to showcase my creative and artistic approaches to programming


Tic-Tac-Toe

Recreated the classic game tic tac toe for the computer using ansi C for a school assignment. You can play against the computer or against a friend. It utilizes hexadecimal to decimal conversions, as well as binary to input the 'X' and 'O' onto your playing board.
